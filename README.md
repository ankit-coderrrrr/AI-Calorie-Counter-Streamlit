# AI Visual Calorie Counter 🍎
An intelligent nutrition tracking application built with Python and Streamlit.

## ✨ Features
- **AI Vision**: Uses Google Gemini 3 Flash to identify food from images.
- **Automated Tracking**: Estimates calories and portions instantly.
- **Data Analytics**: Uses **Pandas** to manage daily meal history in a tabular format.
- **Persistence**: Saves data locally to CSV for long-term tracking.

## 🛠️ Tech Stack
- **Language**: Python 3.14
- **Framework**: Streamlit
- **AI**: Google Generative AI (Gemini API)
- **Data**: Pandas, JSON, Regex
